# 表格

#### 写法
```
    | Tables        | Are           | Cool  |
    | ------------- |:-------------:| -----:|
    | col 3 is      | right-aligned | $1600 |
    | col 2 is      | centered      |   $12 |
    | zebra stripes | are neat      |    $1 |
```
### 显示
| Tables        | Are           | Cool  |
| ------------- |:-------------:| -----:|
| col 3 is      | right-aligned | $1600 |
| col 2 is      | centered      |   $12 |
| zebra stripes | are neat      |    $1 |

# 粗体和斜体
### 写法
```
* **两个\*号包含一端文本就是粗体**
* __两个\_包起来的也是粗体__
* ~~删除线~~
```
### 显示
* **两个\*号包含一端文本就是粗体**
* __两个\_包起来的也是粗体__
* ~~两个\~包起来的就是删除~~
